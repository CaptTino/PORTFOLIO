📚 NLP · Skripsi
Indonesian Book Genre Classification – CNN + FastText + MI
Undergraduate thesis project: an NLP system that automatically classifies Indonesian-language books into 4 genres (Fantasi, Romance, Horor, Misteri) based on their synopsis. Pipeline combines FastText word embeddings for morphologically rich Indonesian text, a CNN architecture for deep feature extraction, and Mutual Information for statistical feature selection — retaining only the most genre-discriminative vocabulary. Trained on a curated dataset of 1,128 Indonesian book synopses and deployed as an interactive Streamlit web app where users input a synopsis and instantly receive a predicted genre with confidence score.
